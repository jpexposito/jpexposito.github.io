# GitHub Pages — Publicar documentación sin perder enlaces

## Reglas de oro

- Usa enlaces **relativos**.
- Crea siempre un `index.md` por carpeta.
- Si cambias rutas, crea wrappers en `legacy/`.

## Estructura recomendada

- `index.md` en la raíz
- contenido organizado por secciones (teoría/prácticas)
