# Práctica 01 — Gestor de tareas (Angular moderno)

## Objetivo
Crear una aplicación Angular con varias páginas (routing), estilos y un formulario para **crear** y **eliminar** tareas (CRUD básico en memoria).

## Tecnologías
- Angular (standalone)
- Angular Router
- Reactive Forms
- TypeScript
- CSS
